Glosses

Interpretive definitions that frame concepts through a broader lens.

  • Atomic Node -- A single markdown file resolving to a wikilink target — the default node shape the graph uses
  • Compound Node -- A folder with a designated lead where the concept has supporting material; the folder resolves to the lead, and sub-files serve that one concept
  • Crescent -- The non-overlap arc in a Venn-style predicate adjacency — what this predicate holds that its neighbor does not
  • Deep Context Community -- The collective of contributors to the Deep Context practice and graph — the default author identity under which nodes are attributed
  • Form Contract -- A specification a node conforms to — the mechanism by which node shapes are named, checkable, and pluralizable
  • Ghost Link -- A wikilink to a target that does not yet exist — a planning signal carried inside the graph itself
  • Markdown Node -- A markdown file read as a graph node — wikilinks and named edges are structure, not decoration
  • Named Edge -- A predicate-annotated wikilink expressing a typed relation — the structural unit of the graph's connection layer
  • Synpraxis -- Coined term for the full spectrum of acting together — coordination, cooperation, collaboration, and the anti-patterns that shadow each
